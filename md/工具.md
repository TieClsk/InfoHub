
# 推荐 Skills 与核心工具

## Skill

本项目主要使用一个 Skill：

**frontend-design**（`/mnt/skills/public/frontend-design/SKILL.md`）

Claude Code 内置，不需要安装。在 Claude Code 中开发前端时说：

> "请先读取 /mnt/skills/public/frontend-design/SKILL.md，然后帮我创建 XXX 页面"

后续如需导出功能，可引入 docx / pdf / xlsx skill。

## DeepSeek 处理管线

```
采集完成
  ↓
按板块批量取当日 RawContent（未处理的）
  ↓
每 5-10 条发给 DeepSeek：去重 + 评分 + 摘要 + 分类 + 翻译
  ↓
解析 JSON 响应，写入 ProcessedContent
  ↓
前端从 ProcessedContent 读取，按 importance 排序展示
```

每周一额外触发周报生成：取本周 importance ≥ 7 的内容 → DeepSeek 生成周报 → 写入 WeeklySummary。
